# Photo Placeholders

Add your anniversary photos here!

## Required Photos for Carousel:
- `photo1.jpg` - First Meeting
- `photo2.jpg` - Wedding Day  
- `photo3.jpg` - Building a Home
- `photo4.jpg` - Growing Family
- `photo5.jpg` - Celebrations
- `photo6.jpg` - Today & Tomorrow

## Tips:
- Use JPEG or PNG format
- Recommended size: 1920x1280 pixels (or any 4:3 ratio)
- Keep file sizes under 2MB for faster loading
- Make sure photos are horizontally oriented

The carousel will show beautiful placeholder gradients until you add your photos!
