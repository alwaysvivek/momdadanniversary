# Music Instructions

## Adding Your Anniversary Song

Place your song file at:
```
public/anniversary-song.mp3
```

## Supported Formats:
- MP3 (recommended)
- WAV
- OGG

## Tips:
- Make sure the file is named exactly `anniversary-song.mp3`
- Keep file size reasonable (under 10MB recommended)
- Test the audio file to ensure it plays correctly
- The player will loop the song automatically

If the file is not present, the music player will show but won't play anything until you add your song!
